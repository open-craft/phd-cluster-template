# Pull Request Sandboxes

You may want to automatically create sandbox instances for GitHub Pull Requests. To do so, the [PR sandbox automation](https://github.com/open-craft/pr-sandbox-automation) must be installed for the given GitHub organization and configured for the cluster.

You can find detailed instructions in the PR sandbox automation's README.
